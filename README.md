# No2ndBest

An iOS game project created with SpriteKit.

## Description

This game is currently in the early development stage.

## Requirements

- Xcode 14.0+
- iOS 16.0+
- Swift 5.0+

## Installation

1. Clone this repository
2. Open No2ndBest.xcodeproj in Xcode
3. Build and run the project
